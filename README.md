# UI Components Demo (PySide6)

## Como rodar
1) Crie um venv e instale PySide6:
```bash
python -m venv .venv
# Windows:
.venv\Scripts\pip install PySide6
# Linux:
.venv/bin/pip install PySide6
```

2) Rode o demo:
```bash
python -m app.main
```
